# Passio docs

Passio is a local-first, passionfruit-themed desktop AI assistant. A floating bubble lives in your corner, remembers what matters, runs autonomous loops, integrates Obsidian, and grows via Seeds (plugins).

## Quick reference

- **Toggle bubble:** `Super+B`
- **Quick chat:** `Super+Space`
- **Global spotlight:** `Super+/`
- **Screenshot-and-ask:** `Super+Shift+S`
- **What-next picker:** `Super+Shift+W`
- **Clipboard-ask:** `Super+Shift+C`
- **Rewrite selection:** `Super+Shift+R`
- **Translate selection:** `Super+Shift+L`
- **Force scan:** `Super+Shift+N`
- **PTT (push-to-talk):** `Super+Alt+Space`
